This project is an interactive **Spotify analytics dashboard** built in **Power BI** to explore listening behavior, artist trends, and track performance. It provides insights into song distribution, popularity patterns, and listening habits over time.

The dashboard is designed to be visually engaging and easy to navigate, enabling quick exploration of key music metrics.

---

## 📊 Features

### 🔍 Summary Metrics

* **Total Songs:** 789
* **Total Artists:** 342
* **Total Listening Time:** 3.28 (unit depends on dataset)
* **Average Popularity Score:** 90

---

### 🎤 Artist Insights

* Top artists ranked by number of songs
* Scrollable artist list for quick filtering
* Highlights dominant artists (e.g., Taylor Swift, Travis Scott, Drake)

---

### 🎵 Song Analytics

* Songs by popularity (top tracks based on Spotify metrics)
* Songs by album type (single vs album comparison)
* Explicit vs non-explicit content distribution

---

### 📈 Time-Based Trends

* Songs by year (track additions over time)
* Average popularity by month (seasonal trends)
* Distinct songs by month (listening diversity)

---

### 📀 Now Playing Panel

* Featured track display with album art
* Spotify-style playback UI for visual appeal

---

### 🎛️ Filters & Navigation

* Toggle between:

  * Overview
  * Artists
  * Songs
* Cross-filtering and drill-down exploration

---

### 🎨 Design Highlights

* Dark Spotify-inspired theme
* Card-based clean layout
* Interactive charts and slicers
* Balanced visual storytelling

# 🎤 Artists Page

<p align="center">
  <img width="1277" height="712" alt="Artists Dashboard"

The **Artists** page focuses on analyzing performance at the artist level within the Spotify dataset.

### Key Insights:

* Number of songs per artist
* Total accumulated popularity
* Frequency of top chart positions

Users can quickly identify dominant artists and compare their contributions.

### Additional Details:

An interactive table enhances exploration by showing:

* Track-level details
* Release dates
* Album types
* Popularity metrics
* Duration

With filtering and drill-down options, this page enables deeper analysis of artist performance over time.

* Album type
* Song count

Dynamic filters allow users to drill into specific songs or artists for deeper insights.
